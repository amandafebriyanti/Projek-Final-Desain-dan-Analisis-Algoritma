# Aplikasi Pencarian Judul Skripsi Berbasis String Matching

Aplikasi ini merupakan sistem pencarian judul skripsi pada database teks yang dibangun menggunakan metode **String Matching**. Aplikasi bertujuan untuk membantu mahasiswa dan pihak akademik dalam melakukan pencarian judul skripsi secara cepat, terstruktur, dan efisien berdasarkan kata kunci yang dimasukkan pengguna.

Dalam pengelolaan data skripsi, pencarian judul secara manual sering kali memerlukan waktu yang lama dan kurang efisien, terutama ketika jumlah data semakin bertambah. Oleh karena itu, dibutuhkan sebuah sistem pencarian yang mampu mencocokkan kata kunci dengan data judul skripsi yang tersimpan dalam database secara otomatis.

Metode **string matching** digunakan sebagai pendekatan sederhana namun efektif untuk mencocokkan pola teks antara input pengguna dan judul skripsi yang tersedia.
